# Hiragana Practice Tool

This is a very basic app to practice the recognition of hiragana characters from the Japanese language.

## About Hiragana
Hiragana is one of three Japanese writing systems. Each hiragana character represent a syllable. We are not dealing with an alphabet but with a phonetic writing system. It is widely used for writing native Japanese words as well as writing particles and indicate grammatical information in a sentence. 

If you want to learn more about Hiragana (and the other writing systems) visit [Hiragana, Katakana, Kanji: The Three Infamous Japanese "Alphabets"](https://ai.glossika.com/blog/hiragana-katakana-kanji).

# About the app

- 46 hiragana characters to practice.
- A message will show the mistakes made and the correct answer at the end.
